# Daily Task Planner

A modern task management application built with Next.js. The app allows users to organize tasks into lists, add labels, set priorities, deadlines, and track time estimates.

## Core Features
- Task organization with lists and labels
- Subtasks and task hierarchy
- Due dates, deadlines, and time tracking
- Priority levels
- Recurring tasks
- File attachments
- Reminders
- Task history/audit trail
- Dark/light theme support
